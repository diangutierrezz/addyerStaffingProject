Instrucciones

Paso a paso para crear un Nuevo Colaborador.

Paso 1: Agregar datos personales

En este paso se deben llenar los campos con los datos personales del nuevo colaborador:

  • Designar Rol (Administrador o Colaborador) del nuevo usuario.

  • Nombre.

  • Apellido.

  • Correo electrónico.

  • Contraseña.

  • Cargo (Mentor, Tutor, Cordinador, Ayudante, Profesor de habilidades blandas, Profesor de programación).

Luego se presiona el botón "Siguiente" que lo llevará al segundo paso.

Paso 2: Agregar Habilidades

Luego deben agregarse las habilidades que posee el colaborador, para ello se deben seleccionar una por una y presionar el botón "Agregar". Al presionar el botón "Finalizar" serás redirigido a una página donde podrás ver todos los colaboradores creados.
