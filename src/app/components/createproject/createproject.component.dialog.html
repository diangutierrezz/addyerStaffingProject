Instrucciones

Paso a paso para crear un Nuevo Proyecto.

Paso 1. Crear Proyecto

En este paso se deben llenar los campos con los datos del nuevo proyecto:

  • Nombre del proyecto.

  • Descripción.

  • Fecha de inicio.

  • Fecha de fin.

Luego se presiona el botón "Siguiente" que lo llevará al segundo paso.

Paso 2. Agregar Habilidades

Luego deben agregarse al proyecto las habilidades que serán requeridas en el mismo, para ello se deben seleccionar una por una y presionar el botón "Agregar". Luego de completado este paso se presiona el botón "Siguiente".

Paso 3: Agregar Colaboradores

En el último paso se deben agregar los coloborares que trabajarán en el nuevo proyecto, estos deben ser agregados según su cargo o habilidad, para ello se puede hacer una búsqueda por estos dos criterios, y para vincularlos se debe presionar el botón "Agregar". Al presionar el botón "Finalizar" serás redirigido a una página donde podrás ver todos los proyectos creados.
